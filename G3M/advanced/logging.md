# Logging

G3M writes logs to the user data directory so problems can be inspected after the fact.

## Main files

- `logs/g3m.log`
- `logs/shortcut.log`

## Archived logs

Older session logs are kept under:

`logs/g3m/`

## When to check logs

Logs are the first place to look for:

- import failures
- patching failures
- download issues
- plugin errors
- launch and restore problems
