# Theme Packages

Theme packages let you import or export a bundle of appearance settings instead
of changing everything by hand.

## Main manifest

The current primary manifest name is:

`theme_config.json`

Legacy `theme.json` is still recognized during import.

## What a theme package can carry

- colors
- border radius
- background file
- logo
- font
- startup sound
- background music
- related appearance flags such as background or animation toggles

## Why use it

It is the fastest way to move a full look and feel between two G3M setups.
