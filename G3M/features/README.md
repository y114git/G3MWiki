# Features

G3M includes a wide set of features beyond basic mod management. This section covers each one in detail.

- [**Profiles**](profiles.md) — Separate mod collections with independent settings.
- [**Downloads**](downloads.md) — Download queue, history, and auto-install system.
- [**Game Versions**](game-versions.md) — Save and restore game file snapshots.
- [**Desktop Shortcuts**](shortcuts.md) — Launch a specific mod configuration without opening G3M.
- [**Chat**](chat.md) — Built-in community chat.
- [**Plugins**](plugins.md) — Extend G3M with third-party plugins.
- [**Blocklist Manager**](blocklist.md) — Hide unwanted mods from the browser and library.
- [**Modding Tools**](modding-tools.md) — Create, apply, merge, compare, and convert patches.
- [**Announcements & Polls**](announcements.md) — In-app news and community polls.
- [**Update System**](update-system.md) — Automatic and manual update checking.
- [**Online Presence**](online-presence.md) — Player counter in the title bar.
- [**Plugin Development Guide**](plugins-development.md) — Build plugins against the current plugin API.
