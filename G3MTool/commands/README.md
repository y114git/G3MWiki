# Commands Overview

| Command | Description |
| --- | --- |
| [`patch`](patch.md) | Create, apply, validate, or merge `.g3mpatch` files |
| [`xpatch`](xpatch.md) | Create or apply xdelta patches |
| [`execute`](execute.md) | Run `.csx` scripts, invoke bundled xdelta, or launch an external program |
| [`info`](info.md) | Print data-file or `.g3mpatch` metadata |
| [`diff`](diff.md) | Compare data files and/or `.g3mpatch` files |
| [`--version`](version.md) | Print the installed G3MTool version |

All data-file arguments accept `.win`, `.ios`, `.unx`, and `.droid`.

Global options are documented in [Getting Started](../getting-started.md#global-options). Some commands also support `--cache <dir>` for reusable `.g3mcache` analysis.
